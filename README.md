# **K-POP Multiple Choice (K-POP 뮤비 퀴즈 : 객관식 인기 뮤비 맞히기)**

기존의 KPOP 뮤비 퀴즈를 객관식으로 출시한 게임입니다. 포인트를 이용해 다양한 모드로 진행이 가능하며 기존과는 다르게 터치로 UI 조작이 가능합니다. Google 로그인 및 Google Play Games 서비스를 이용해 유저들과 경쟁할 수 있습니다.
